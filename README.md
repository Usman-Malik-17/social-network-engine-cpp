# Social Network Engine (C++)

A console-based social networking application built in C++ using Object-Oriented Programming concepts.  
The project simulates basic features of a social networking platform such as user accounts, friendships, and posts.

## Features

- Create and manage user accounts
- Add and manage friend relationships
- Create and view posts
- Display user timelines
- File-based data storage for persistent data

## Technologies Used

- C++
- Object-Oriented Programming (OOP)
- File Handling

## Learning Objectives

This project was developed to practice:

- Object-Oriented Programming concepts
- Designing modular C++ applications
- File handling for persistent data storage
- Building a menu-driven console application

## Sample Output

The application runs as a menu-driven console program allowing users to create accounts, add friends, and interact with posts.
